\doxysubsection{Detailed Description}
A class representing a light-\/emitting entity in a 3D environment. 

The \doxylink{classLightEmitter}{Light\+Emitter} class inherits from the \doxylink{classEntity}{Entity} class and represents a light source in the game world. It provides methods to get the light space matrix, light projection, and light view.
