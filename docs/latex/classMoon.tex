\doxysubsection{Detailed Description}
A class representing a \doxylink{classMoon}{Moon} entity that emits light and rotates around a point. 

The \doxylink{classMoon}{Moon} class inherits from the \doxylink{classLightEmitter}{Light\+Emitter} class and represents a moon object in the scene. It handles the moon\textquotesingle{}s position, size, rotation, and light emission properties.
