\doxysubsection{Detailed Description}
Represents a rocket entity in the simulation. 

The \doxylink{classRocket}{Rocket} class inherits from the \doxylink{classEntity}{Entity} class and represents a rocket that can move towards a target and explode upon reaching it, triggering a particle system.
